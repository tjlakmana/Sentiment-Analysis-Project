# Sentiment-Analysis-Project
## Twitter/X Stock Sentiment Analysis Project
Project Overview

This project focuses on collecting and analyzing unstructured stock-related text data from Twitter/X and financial news sources. The system will use Natural Language Processing (NLP) and sentiment analysis techniques to identify market sentiment trends related to stock tickers and financial discussions.

Project Objectives
Collect stock-related tweets and financial news data using APIs and RSS feeds
Preprocess and clean unstructured text data
Extract stock tickers and company entities from text
Perform sentiment analysis using FinBERT
Aggregate sentiment trends and analytics by ticker and time window
Visualize sentiment trends and analytics through a real-time dashboard
